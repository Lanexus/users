# DeepValidator V27

SSH/FTP Credential Validator & Domain Scanner

## Features

- SSH/FTP credential validation
- Domain enumeration
- Discord webhook notifications
- Auto-install dependencies
- Smart file parsing
- TLD/Domain filtering
- Batch processing for large files

## Installation

```bash
git clone https://github.com/Lanexus/deep_new.git
cd deep_new/lab
python3 loader.py
```

## Usage

```bash
# Interactive mode
python3 loader.py

# CLI mode
python3 loader.py -f url -o res -t 500 --use-ip --pass YOUR_PASSWORD

# With TLD filter
python3 loader.py -f url -o res --tld id,com,net --pass YOUR_PASSWORD
```
